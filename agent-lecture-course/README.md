# 从 ChatGPT 到 Agent：AI 智能体入门讲座课程库

## 课程定位

这是一套 3 次讲座、每次约 2 小时的入门课程，主线不是“炫工具”，而是帮助学习者建立对 Agent 的基本判断力：

- Agent 和传统聊天式 AI 到底差在哪里
- OpenClaw 这一类工具为什么值得关注
- 如何完成一次基础部署与配置
- 如何把 Agent 用在学习辅助里，而不是把脑子外包给它

整套课程采用“认知先行，部署跟上，应用落地”的设计思路。

## 课程目标

### 认知目标

- 理解聊天 AI 与 Agent 的核心差异
- 理解 Agent 的基本构成：模型、工具、记忆、渠道、权限
- 理解 OpenClaw 类工具的价值、边界与典型风险

### 技能目标

- 完成一次可运行的 Agent 基础部署或配置
- 能下达较清晰的任务指令
- 能观察 Agent 的执行过程并识别明显问题

### 素养目标

- 形成“结果要核查”的使用意识
- 形成“权限不能乱给”的安全意识
- 形成“AI 用来辅助学习，不是替代成长”的使用底线

## 三讲结构

| 讲次 | 主题 | 核心问题 | 课堂产出 |
|------|------|----------|----------|
| 第 1 讲 | 认识 Agent | Agent 为什么不只是更会聊天？ | 能说清聊天 AI 与 Agent 的区别 |
| 第 2 讲 | 部署与配置 | Agent 怎样真正跑起来？ | 完成一次基础部署/配置并运行简单任务 |
| 第 3 讲 | 学习应用与工具比较 | 怎样让 Agent 帮忙，但不帮倒忙？ | 能做基础选型，能判断使用边界 |

## 讲课风格建议

这套课适合讲得像“拆神话现场”，不要讲成产品发布会，也不要讲成纯技术培训。

- 用比喻代替抽象定义
- 用对比代替堆概念
- 用真实任务代替空洞示例
- 用一点幽默感降低技术距离感

推荐统一比喻：

- 聊天 AI 像“嘴很快的顾问”
- Agent 像“能动手的助理”
- 模型像“大脑”
- 工具像“手脚”
- 记忆像“备忘录”
- 权限像“办公室钥匙”
